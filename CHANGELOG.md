# Changelog

## Milestone 1 + OneMap Foundation

- New clean workflow and responsive interface.
- Correct mass-import separation for Bed, Initials and Postal Code.
- Bed format uses letter-number without dash, e.g. A12.
- Visit preparation with tasks, free text, custom duration and timing constraints.
- Nurse start/end options: JMC, NUH or Home.
- Day and PM shift rules, lunch/dinner blocks and fifth-short-visit warning.
- Stable reset, strike-off, replacement and nurse-specific ad hoc workflows.
- Autosave and allocation persistence.
- Vercel server functions for OneMap authentication, postal search and driving routes.
- Finalise, print/PDF and CSV export.
